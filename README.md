# Automatización de Reportes de Logística e Incidentes 

Un script automatizado en **Python** enfocado en la limpieza, validación y procesamiento de datos sobre incidentes operativos y rescates críticos de transporte. Este proyecto une la lógica de desarrollo de software con la experiencia en coordinación de operaciones en tiempo real.

## Funcionalidades Principales
* **Validación y Limpieza de Datos:** Identifica y descarta automáticamente registros corruptos o incompletos (filas sin número de unidad) simulando flujos JSON/Excel.
* **Filtrado Operativo Remoto:** Procesa grandes volúmenes de datos en segundos para clasificar incidencias según su nivel de prioridad.
* **Consolidación de Métricas:** Genera un reporte resumido y estructurado en consola con los indicadores clave (KPIs) de rescates coordinados con éxito.

## Habilidades Técnicas Demostradas
* Modelado y manejo de estructuras de datos complejas (Listas de Diccionarios).
* Automatización de tareas administrativas mediante scripts lógicos.
* Resolución de problemas bajo un enfoque técnico-operativo.
* Gestión de flujos de trabajo orientados a la optimización de procesos remotos.
